Etiquetas y Estructura

HTML (HyperText Markup Language) es el esqueleto de toda página web. Define el contenido y su estructura mediante etiquetas.

¿Qué es HTML?

Un documento HTML está formado por elementos que consisten en una etiqueta de apertura, contenido y una etiqueta de cierre. El navegador interpreta estas etiquetas y renderiza la página.

💡

Anatomía de una etiqueta

<etiqueta atributo="valor">Contenido</etiqueta> — apertura, atributos opcionales, contenido y cierre.

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <title>Mi primera página</title>
  </head>
  <body>
    <h1>¡Hola Mundo!</h1>
    <p>Bienvenido a mi sitio web.</p>
  </body>
</html>

Etiquetas esenciales

Estas son las etiquetas que usarás en casi todos los proyectos:

<h1> – <h6>Encabezados de nivel 1 al 6
<p>Párrafo de texto
<a>Enlace / hipervínculo
<img>Imagen (etiqueta vacía)
<ul> / <ol>Listas no ordenadas / ordenadas
<li>Elemento de lista
<div>Contenedor genérico de bloque
<span>Contenedor genérico en línea
<strong>Texto en negrita (importante)
<em>Texto en cursiva (énfasis)
<br>Salto de línea
<hr>Línea horizontal separadora
<table>Tabla de datos
<form>Formulario interactivo
<input>Campo de entrada de datos
<button>Botón interactivo

Ejemplo: Listas y enlaces

<ul>
  <li>Elemento uno</li>
  <li>Elemento dos</li>
</ul>

<a href="https://google.com" target="_blank">
  Ir a Google
</a>

<img src="foto.jpg" alt="Descripción" width="300">

HTML Semántico

Las etiquetas semánticas describen el significado del contenido, mejorando la accesibilidad y el SEO. Usa <div> solo cuando ninguna otra etiqueta semántica aplique.

<header>Cabecera del sitio o sección
<nav>Menú de navegación
<main>Contenido principal único
<section>Sección temática del contenido
<article>Artículo independiente
<aside>Contenido secundario / barra lateral
<footer>Pie de página
<figure>Imagen con leyenda opcional
<body>
  <header><nav>...</nav></header>
  <main>
    <section>
      <article>...</article>
    </section>
    <aside>...</aside>
  </main>
  <footer>...</footer>
</body>

Formularios

Los formularios permiten al usuario ingresar y enviar datos. Cada campo <input> tiene un atributo type que define qué tipo de dato acepta.

<form action="/enviar" method="POST">

  <label for="nombre">Nombre:</label>
  <input type="text" id="nombre" name="nombre" required>

  <label for="email">Email:</label>
  <input type="email" id="email" name="email">

  <select name="pais">
    <option value="co">Colombia</option>
    <option value="mx">México</option>
  </select>

  <textarea name="mensaje" rows="4"></textarea>

  <button type="submit">Enviar</button>
</form>
